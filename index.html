This site serves as a dogfooding ground for site2skill-go, verifying crawler behavior with real-world complexity.

It intentionally exposes ambiguous and conflicting locale URL patterns to test robustness.

URL Patterns

Pattern A: Path-based Locale

  • /docs/ja/getting-started/ - Japanese
  • /docs/en/getting-started/ - English
  • /docs/zh-tw/getting-started/ - Traditional Chinese

Locale is explicitly embedded in the path. These pages return 200 OK.

Pattern B: Query-based Locale

  • /docs/pattern-b/?lang=ja - Japanese
  • /docs/pattern-b/?lang=en - English
  • /docs/pattern-b/?lang=zh-tw - Traditional Chinese

Path is identical. Locale is specified via query parameter. Single page with JavaScript-based locale switching.

Pattern C: Mixed Locale (Conflict)

  • /docs/ja/pattern-c/?lang=en - Japanese path + English query
  • /docs/en/pattern-c/?lang=ja - English path + Japanese query
  • /docs/zh-tw/pattern-c/?lang=en - Traditional Chinese path + English query

Path locale and query locale conflict. Path locale MUST win. Each page detects and displays the conflict.

Pattern D: No Locale

  • /docs/getting-started/

No locale hint in path or query.

Pattern E: Varied Availability

  • /docs/ja/installation/ - Japanese Only
  • /docs/en/api-reference/ - English Only
  • /docs/ja/examples/, /docs/en/examples/ - Ja/En Only (No zh-tw)

Pages that exist only in specific locales.

ja missing, en exists Only en is fetched

site2skillgo Usage

Basic Command

site2skillgo generate <URL> <SKILL_NAME> [options]

Locale Priority Options

Option Description
--locale-priority Locale priority order (default "en,ja")
--no-locale-priority Disable locale priority mode
--locale-param Query parameter name for locale (e.g., "hl")

Examples

# Crawl with Japanese priority
site2skillgo generate --locale-priority "ja,en" https://f4ah6o.github.io/site2skill-go/ myskill

# Use query parameter for locale
site2skillgo generate --locale-param "hl" https://f4ah6o.github.io/site2skill-go/ myskill

# Disable locale priority
site2skillgo generate --no-locale-priority https://f4ah6o.github.io/site2skill-go/ myskill

Notes

  • No redirects are intentionally used
  • Canonical URLs may exist in multiple forms
  • This site is intentionally hostile to naive crawlers
