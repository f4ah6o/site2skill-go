What is site2skill-go?

site2skill-go is a tool that crawls documentation websites and converts them into skill packages for Claude and Codex agents.

Installation

go install github.com/f4ah6o/site2skill-go/cmd/site2skillgo@latest

Basic Usage

# Generate a Claude skill
site2skillgo generate https://docs.example.com myskill
